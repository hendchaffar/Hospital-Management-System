# Hospital Management System

## Description
Hospital management system built with Django for managing doctors, patients and appointments through an admin dashboard.

## Features
- Admin authentication
- Doctor management
- Patient management
- Appointment management
- CRUD operations for doctors, patients and appointments

## Tech Stack
- Python
- Django
- HTML
- CSS
- SQLite

## Purpose
This project demonstrates my ability to build a database-driven web application using Django with authentication, CRUD operations and basic administrative workflows.

## Main Functionalities
- Add, view and delete doctors
- Add, view and delete patients
- Add, view and delete appointments
- Dashboard with summary counts
